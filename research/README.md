# research/

Captured research and independent verification on the Siri Bhoovalaya manuscript and this
decoder.

| File | What it is |
|---|---|
| [`siri-bhoovalaya-research.md`](./siri-bhoovalaya-research.md) | Readable deep-research report — history, the chakra grid, **all the bandhas**, the multilingual/718-languages claim, decipherment history, computational efforts. Confidence tags from a 3-vote adversarial verifier. |
| [`verification-notes.md`](./verification-notes.md) | Independent hands-on checks against this repo: Chakra-Bandha reproduced from scratch (729/729), all 9 chakras decoded, mapping confirmed against `script.ts`. |
| [`siri-bhoovalaya-research-raw.json`](./siri-bhoovalaya-research-raw.json) | Raw harness output: every verified claim with per-claim vote (e.g. `3-0`, `2-1`), evidence quotes, source URLs, source-quality ratings, refuted claims, and run stats. |

**Method:** fan-out web search across 5 angles → fetch 16 sources → extract 69 claims →
3-vote adversarial verification (2/3 refutes to kill) → synthesize. 23 of 25 verified
claims confirmed; 2 refuted (recorded in the JSON under `refuted`).

**Headline finding, cross-checked by hand:** the literature's Chakra-Bandha (start
Row 1 / Col 14) matches this repo's `chakra_bandha.txt` (`0,13`) exactly, and reproduces
the attested *maṅgala* verse deterministically — the reading comes from the key, not the
dictionary. The field's real frontier is the **six named-but-undocumented bandhas** and the
**unreproduced multilingual layers**.
